# he_music_flutter

A new Flutter project.

## 常用命令

项目根目录已经提供 `Makefile`，以后常用命令直接执行：

```bash
make help
make get
make run
make analyze
make test
make gen
make build-aab
```

常见发布命令：

```bash
make build-apk
make build-aab
```
